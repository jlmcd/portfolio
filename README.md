# Portfolio

This is Jonathan McDonald's personal portfolio website, built with React and Vite.

## Development

Install dependencies and start the development server:

```bash
npm install
npm run dev
```

Open [http://localhost:5173](http://localhost:5173) in your browser.

## Build

Generate a production build:

```bash
npm run build
```

## Deployment

Deploy to platforms like Netlify or Vercel. Ensure all routes fallback to `index.html` (e.g., via a `_redirects` file).

## Backend Server (Optional)

If using the Express backend, add a `.env` file in the project root:

```env
SERVER_PORT=4000
```

Start the backend:

```bash
node server/server.js
```
